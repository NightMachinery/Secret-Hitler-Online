# Investigations

When an investigation power resolves, the investigating president privately learns the target player's party membership:

- Liberal targets show as **LIBERAL**.
- Fascist and Hitler targets show as **FASCIST**.
- Anarchist targets show as **ANARCHIST** when the active setup reveals Anarchists to investigations.

The result alert is private to the investigator. Other players only see that an investigation happened.

## History panel visibility

If the lobby has the history panel and **Show presidential actions** enabled, investigation actions appear in the history panel.

- The investigating president sees the private result on their own history row, for example: `Alice investigated Bob — result: Fascist`.
- The target and other players see only the public action, for example: `Alice investigated Bob`.
- If presidential actions are disabled, investigation actions and private results are hidden from history.

Observers controlling a player's seat see the same private investigation history that the controlled seat would see.
